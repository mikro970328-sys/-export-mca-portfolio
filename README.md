# EXPORT MCA LLC Portfolio Website

Sitio estático de una sola página.

## Publicarlo en GitHub Pages
1. Cree un repositorio nuevo llamado `export-mca-portfolio`.
2. Suba los archivos de esta carpeta.
3. En GitHub: Settings > Pages > Deploy from a branch > `main` / root.
4. Conecte su dominio después desde la configuración de Pages o su proveedor de hosting.

## Antes de publicar
- Cambie el enlace de WhatsApp por su número real en `index.html`.
- Revise cualquier categoría de producto antes de usarla comercialmente.
